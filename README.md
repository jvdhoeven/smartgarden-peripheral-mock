# SmartGarden Mock Peripheral

## Prequisites
- NodeJS (v12 - v14) / NPM
- Install prerequisites from Bleno! https://github.com/abandonware/bleno#prerequisites

## Installation

```
npm install
```

## Run

```
npm run start
```
